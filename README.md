# lofi-girl-web-app

A simple webpage with all of [Lofi Girl](https://lofigirl.com/)'s radios accessible via a Youtube embed.

Accessible for free, forever\*, at [lofi.valbuilds.xyz](https://lofi.valbuilds.xyz).

> \* Or as long as I keep it up :3
